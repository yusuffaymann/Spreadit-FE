Class: Toogle

Toogle(optionTitle, optionDescription, isToggled, onToggle, disabled) → {JSX.Element}

new Toogle(optionTitle, optionDescription, isToggled, onToggle, disabled) → {JSX.Element}

Component for displaying the toogle switch.

Parameters:
Name Type Description
optionTitle string

The title of the switch
optionDescription string

The description of the switch
isToggled boolean

The state of the switch (on or off)
onToggle function

The function to be called upon clicking on the switch [Required*]
disabled boolean

Whether the switch is disabled or not if true the switch becomes unreactive and grayed out
Source:
  • components/UI/Switch.jsx, line 35
Returns:

The rendered Switch component.

Type
JSX.Element
Examples
//renders a toogle with only a title, by default it is enabled and set to off
const title = "Name"
function print() {console.log("switched")}
<Toogle optionTitle={title} onToggle={print()} />
//renders a toogle with a title and a description, by default it is enabled and set to off
const title = "Name"
const description = "Description"
function print() {console.log("switched")}
<Toogle optionTitle={title} optionDescription={description} onToggle={print()} />
//renders a toogle with a title and a description and set to on and disabled
const title = "Name"
const description = "Description"
const toggled = true
const inactive = true
function print() {console.log("switched")}
<Toogle optionTitle={title} optionDescription={description} isToggled={toggled} disabled={inactive} onToggle={print()} />
